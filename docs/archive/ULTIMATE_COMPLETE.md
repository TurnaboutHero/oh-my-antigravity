### 🚀 설치 방법 (초간단!)

이제 **단 한 줄**로 설치 가능합니다:

```bash
# 방법 1: npx로 즉시 실행 (설치 없이)

# 방법 2: 전역 설치 후 사용
npm install -g oh-my-antigravity
oma list

# 방법 3: 프로젝트에 로컬 설치
npm install oh-my-antigravity
npx oma install oracle --project
```

### 🧠 Sisyphus 오케스트레이션 시스템

**마스터 오케스트레이터**: Sisyphus
- 복잡한 요청을 분석하고 전문가 에이전트에게 위임
- 작업 순서와 의존성 관리
- 결과 통합 및 조정

**전문 에이전트 팀** (8명):

1. **Sisyphus** (오케스트레이터)
   - 작업 분석 및 delegation
   - 워크플로우 조정
   
2. **Oracle** (아키텍트)
   - 시스템 설계
   - 기술 스택 선택
   - 성능 최적화 전략

3. **CodeSmith** (코드 장인)
   - 프로덕션 급 코드 작성
   - clean code 원칙
   - SOLID 패턴

4. **Pixel** (UI/UX 전문가)
   - 컴포넌트 디자인
   - 반응형 레이아웃
   - 접근성 준수

5. **Tester** (QA 전문가)
   - 단위 테스트
   - 통합 테스트
   - 80%+ 커버리지

6. **Librarian** (리서치 전문가)
   - API 문서 검색
   - 코드 예제 제공
   - 학습 자료 추천

7. **SecurityGuard** (보안 전문가)
   - OWASP Top 10 검사
   - 취약점 탐지
   - 보안 best practice

8. **Debugger** (버그 헌터)
   - 빠른 이슈 식별
   - 근본 원인 분석
   - 회귀 방지

### 📦 프로젝트 구조

```
oma/
├── package.json           # NPM 패키지 설정

│   ├── install.js         # 설치 스크립트
│   ├── oma.js            # CLI 진입점
│   └── postinstall.js    # 자동 설치
├── plugins/              # 전문 에이전트 스킬
│   ├── sisyphus/         # 마스터 오케스트레이터
│   ├── oracle/           # 아키텍트
│   ├── codesmith/        # 코드 작성
│   ├── pixel/            # UI/UX
│   ├── tester/           # QA
│   ├── librarian/        # 리서치
│   ├── security-guard/   # 보안
│   └── debugger/         # 디버깅
├── bin/                  # 크로스 플랫폼 CLI
├── lib/                  # 코어 라이브러리
├── commands/             # 슬래시 명령어
├── themes/               # 에이전트 테마
└── extension/            # VS Code GUI

```

